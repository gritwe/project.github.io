Сгибания пальцев с эспандером

Простой
Эспандеры
Предплечья, кисти

Описание

Сгибания пальцев с эспандером — изолирующее упражнение для укрепления мышц предплечий, кистей и пальцев. Особенно полезно для людей, чья деятельность связана с тонкой моторикой рук, а также для спортсменов, желающих улучшить силу хвата.

Техника выполнения

1. Сядьте на стул или встаньте прямо.
2. Возьмите резиновый эспандер в одну руку.
3. Расположите эспандер так, чтобы он проходил через ладонь и пальцы.
4. Четыре пальца (указательный, средний, безымянный, мизинец) лежат на одной стороне эспандера.
5. Большой палец находится на другой стороне для фиксации.
6. На выдохе сжимайте пальцы, преодолевая сопротивление эспандера.
7. Сожмите пальцы полностью, максимально напрягая мышцы предплечья.
8. На вдохе медленно разожмите пальцы, контролируя движение.

Советы

✓ Начинайте с эспандера слабого сопротивления
✓ Выполняйте движение медленно и подконтрольно
✓ Не помогайте себе другой рукой
✓ Фокусируйтесь на работе мышц предплечья
✓ Можно выполнять на каждую руку отдельно
✓ Для прогрессии используйте эспандеры с большим сопротивлением

Оборудование

Резиновый эспандер для рук, кистевой эспандер

Преимущества

Укрепление мышц предплечий и кистей, улучшение силы хвата, профилактика травм запястий, развитие мелкой моторики

Время отдыха

45–60 секунд между подходами

Безопасность

Избегайте чрезмерного напряжения, не тренируйтесь при боли в запястьях, используйте адекватное сопротивление
