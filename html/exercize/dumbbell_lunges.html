Выпады с гантелями

Средний
Гантели
Ноги

Описание

Выпады с гантелями — упражнение для ног и ягодиц с акцентом на баланс. Отлично развивает квадрицепсы, ягодичные мышцы и улучшает координацию.

Техника выполнения

1. Встаньте прямо, ноги на ширине плеч.
2. Возьмите гантели в обе руки, опустите их вдоль тела.
3. Сделайте шаг вперёд правой ногой, перенося вес тела на неё.
4. Опуститесь вниз, сгибая обе ноги до угла 90 градусов.
5. Переднее колено не должно выходить за носок.
6. Заднее колено почти касается пола, но не ложится на него.
7. Оттолкнитесь передней ногой, вернитесь в исходное положение.

Советы

✓ Держите корпус вертикально, не наклоняйтесь вперёд
✓ Не позволяйте колену выходить за носок
✓ Шаг должен быть достаточно широким для хорошей растяжки
✓ Выполняйте одинаковое количество повторений на каждую ногу
✓ Не торопитесь, контролируйте движение в обеих фазах
✓ Начинайте без веса для отработки техники

Оборудование

Гантели

Преимущества

Развитие квадрицепсов и ягодиц, улучшение баланса, симметричная нагрузка

Время отдыха

60–90 секунд между подходами

Безопасность

Не позволяйте колену выходить за носок, используйте умеренный вес, следите за равновесием
