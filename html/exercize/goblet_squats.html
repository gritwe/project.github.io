Приседания с гантелями

Средний
Гантели
Ноги

Описание

Приседания с гантелями — базовое упражнение для ног без штанги. Отлично подходит для домашних тренировок и новичков.

Техника выполнения

1. Возьмите одну гантель двумя руками за один из дисков.
2. Держите гантель вертикально перед грудью.
3. Ноги на ширине плеч, носки слегка развёрнуты наружу.
4. Спина прямая, грудь расправлена, взгляд вперёд.
5. На вдохе опуститесь, сгибая колени и отводя таз назад.
6. Опускайтесь до параллели бёдер с полом или ниже.
7. На выдохе вытолкните себя в исходное положение.

Советы

✓ Держите гантель близко к груди
✓ Не округляйте спину, держите её прямой
✓ Не позволяйте коленям выходить за носки
✓ Опускайтесь медленно, поднимайтесь мощно
✓ Дышите правильно: вдох на опускании, выдох на подъёме
✓ Начинайте с лёгкого веса для отработки техники

Оборудование

Гантель

Преимущества

Развитие ног и ягодиц, безопасность для спины, доступность в домашних условиях

Время отдыха

60–90 секунд между подходами

Безопасность

Используйте умеренный вес, держите спину прямой, не опускайтесь слишком быстро
