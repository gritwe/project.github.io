Отжимания на одной руке

Сложный
Без оборудования
Разное

Описание

Отжимания на одной руке — сложное упражнение для груди, трицепса и плеч, требующее значительной силы и стабильности корпуса. Развивает не только мышцы верхней части тела, но и улучшает баланс и координацию.

Техника выполнения

1. Примите упор лежа, поставив ноги шире плеч для баланса.
2. Одну руку заведите за спину или положите на бедро.
3. Расположите рабочую руку под грудью, не под плечом.
4. На вдохе медленно опуститесь, почти касаясь грудью пола.
5. Тело должно оставаться прямым, без перекоса в стороны.
6. На выдохе мощно отожмитесь в исходное положение.
7. После подхода повторите на другую руку.

Советы

✓ Начинайте с отжиманий с узкой постановкой рук
✓ Тренируйтесь с частичной амплитудой сначала
✓ Используйте стену или наклонную поверхность для упрощения
✓ Держите пресс и ягодицы напряжёнными для стабилизации
✓ Сначала освойте технику с широкой постановкой ног
✓ Не торопитесь, качество важнее количества

Оборудование

Без оборудования, коврик (опционально)

Преимущества

Развитие силы верхней части тела, улучшение баланса и координации, укрепление кора, увеличение стабильности плеч

Время отдыха

90-120 секунд между подходами для каждой руки

Безопасность

Разминка плечевых суставов обязательна, прогрессируйте постепенно, при боли в запястье используйте упоры для отжиманий
