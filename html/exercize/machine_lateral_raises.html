Разведение рук на тренажере

Начальный
Тренажёр
Плечи

Описание

Разведение рук на тренажере — изолированная тренировка для средних пучков плеч. Тренажёр обеспечивает стабильную траекторию движения, что делает упражнение безопасным для новичков.

Техника выполнения

1. Сядьте в тренажёр, плотно прижмите спину к спинке.
2. Возьмитесь за рукоятки, локти согнуты под углом 90-100 градусов.
3. На выдохе разведите руки в стороны, преодолевая сопротивление.
4. В конечной точке сведите лопатки.
5. На вдохе плавно сведите руки перед собой, не бросая вес.
6. Контролируйте движение в обеих фазах.

Советы

✓ Настройте тренажёр под свой рост
✓ Держите спину плотно прижатой к спинке
✓ Не используйте слишком тяжёлый вес
✓ Выполняйте движение плавно, без рывков
✓ В конечной точке делайте паузу 1-2 секунды
✓ Сфокусируйтесь на ощущении работы средних дельт

Оборудование

Тренажёр для разведения рук

Преимущества

Безопасная изоляция средних дельт, стабильная траектория, подходит для новичков

Время отдыха

60–90 секунд между подходами

Безопасность

Настройте тренажёр под себя, не используйте чрезмерный вес, избегайте рывков
